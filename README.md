# Boss直聘批量搜索工具

一个强大的Boss直聘职位信息批量获取工具，支持多种配置生成方式，让你轻松搜索心仪职位！

## 🚀 主要功能

✅ **智能配置生成** - 三种方式生成搜索配置，告别手动编码  
✅ **多城市批量搜索** - 一次配置，多个城市同时搜索  
✅ **灵活筛选条件** - 支持薪资、经验、学历、公司规模等筛选  
✅ **多种输出格式** - JSON/CSV/TXT格式，满足不同需求  
✅ **自动去重过滤** - 智能过滤重复和无效职位  
✅ **简洁中文输入** - 支持"10-20k"、"3-5年"等直观输入

## 📁 文件说明

### 核心文件
- **boss_zhipin.py** - 核心爬虫代码库（不要修改）
- **batch_search.py** - 批量搜索脚本（运行这个执行搜索）
- **batch_config.json** - 批量搜索配置文件
- **login.py** - 登录脚本（首次使用需要运行）

### 配置生成器 🆕
- **config_generator.py** - 交互式配置生成器
- **quick_config.py** - 快速命令行生成器  
- **template_config.py** - 模板配置生成器

### 说明文档
- **配置生成器使用说明.md** - 配置生成器详细使用说明
- **批量搜索使用说明.md** - 批量搜索详细说明

## 🎯 快速开始

### 1. 首次登录
```bash
python3 login.py
```
浏览器会自动打开，微信扫码登录即可。

### 2. 生成配置（三选一）

#### 🔥 方式一：快速生成（推荐）
```bash
# 完整参数
python3 quick_config.py "Python开发@北京,10-20k,3-5年,本科,1000-9999人"

# 部分参数
python3 quick_config.py "前端开发@上海,20-50k"

# 多个任务
python3 quick_config.py "Python开发@北京,10-20k" "Java开发@上海,20-50k"
```

#### 💡 方式二：交互式生成（新手友好）
```bash
python3 config_generator.py
```

#### 🎨 方式三：模板生成（批量搜索）
```bash
python3 template_config.py python_dev 北京,上海,深圳
```

### 3. 执行搜索
```bash
python3 batch_search.py
```

## 🔧 高级功能

### 模板搜索
```bash
# Python开发模板（4个职位 × N个城市）
python3 template_config.py python_dev 北京,上海,深圳,杭州

# 前端开发模板
python3 template_config.py frontend_dev 北京,上海

# 产品经理模板  
python3 template_config.py product_manager 北京,上海,深圳
```

### 自定义设置
- 滚动次数：控制加载职位数量
- 输出格式：JSON/CSV/TXT
- 结果合并：将所有结果合并到一个文件
- 公司黑名单：过滤不想要的公司
- 标签过滤：过滤特定标签的职位

## 📈 输出结果

### JSON格式（默认）
```json
[
  {
    "company": "某科技公司",
    "title": "Python开发工程师", 
    "salary": "15-25K",
    "city": "北京",
    "desc": "职位描述...",
    "url": "https://www.zhipin.com/job_detail/..."
```

### CSV格式（Excel可打开）

格式化的文本输出，便于阅读和打印。
