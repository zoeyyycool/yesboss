# Boss直聘配置生成器使用说明

现在你可以用更简洁的方式生成配置文件，不需要手动查找和匹配各种代码！

## 项目文件说明

### 核心文件
- **boss_zhipin.py** - 核心爬虫代码库（不要修改）
- **batch_search.py** - 批量搜索脚本（运行这个执行搜索）
- **batch_config.json** - 批量搜索配置文件
- **login.py** - 登录脚本（首次使用需要运行）

### 配置生成器
- **config_generator.py** - 交互式配置生成器
- **quick_config.py** - 快速命令行生成器  
- **template_config.py** - 模板配置生成器

### 数据文件
- **cookies.json** - 登录状态文件（自动生成，不要删除）

### 说明文档
- **README.md** - 项目总体说明
- **批量搜索使用说明.md** - 详细的批量搜索说明
- **配置生成器使用说明.md** - 本文档

## 快速开始

### 1. 首次使用：登录
```bash
python3 login.py
```
浏览器会自动打开，使用微信扫码登录即可。

### 2. 生成配置文件（三种方式）

#### 方式一：交互式生成（推荐新手）
```bash
python3 config_generator.py
```
逐步引导输入，支持多任务配置，错误提示友好。

#### 方式二：快速命令行生成（推荐熟练用户）
```bash
python3 quick_config.py "Python开发@北京,10-20k,3-5年,本科,1000-9999人"
```
**格式:** `"职位@城市,薪资,经验,学历,规模"`

#### 方式三：模板生成（推荐批量搜索）
```bash
python3 template_config.py
```
预设常用职位模板，支持多城市批量搜索。

### 3. 运行批量搜索
```bash
python3 batch_search.py
```

## 支持的简化输入

### 城市名称
直接输入中文城市名，无需记忆代码：
```
北京、上海、深圳、杭州、成都、武汉、西安、南京、苏州、天津、重庆、长沙、郑州...
```

### 薪资范围  
支持多种格式：
```
3-5k、5-10k、10-20k、20-50k、50k以上、3k以下
```

### 工作经验
直接输入描述：
```
1年内、1-3年、3-5年、5-10年、10年以上、不限
```

### 学历要求
直接输入学历：
```
大专、本科、硕士、不限
```

### 公司规模（新增）
直接输入规模：
```
0-20人、20-99人、100-499人、500-999人、1000-9999人、10000人以上、不限
```

## 使用示例


- 任务名称: 北京Python开发

- Python开发、Python后端、Django开发、Flask开发
- 在北京、上海、深圳、杭州四个城市

### 示例5: 交互式详细配置

按提示输入：
```

搜索关键词: Python开发
城市: 北京
薪资范围: 10-20k
工作经验: 5-10年

公司规模: 1000-9999人
```

## 常见问题

### Q: 如何添加更多任务？
A: 在交互模式中可以逐个添加，或在命令行模式中用多个引号分隔的任务。

### Q: 如何修改所有任务的共同设置？
A: 修改生成的配置文件中的 `global_settings` 部分。

### Q: 如何只执行部分任务？
A: 可以编辑生成的配置文件，注释掉不需要的任务。

### Q: 参数顺序重要吗？
A: 在快速模式中，参数顺序是：薪资,经验,学历,规模。可以用空值跳过某个参数。

### Q: 如何跳过某个可选参数？
A: 在快速模式中用空值，如：`"Python开发@北京,,3-5年,本科"` (跳过薪资)

| 方式 | 优势 | 适用场景 |
|------|------|----------|
| 手动编辑 | 完全控制 | 复杂自定义需求 |
| 交互式生成器 | 引导友好 | 新手用户 |
| 快速生成器 | 效率最高 | 熟练用户 |
| 模板生成器 | 标准化 | 批量搜索 |

## 注意事项

1. **必需参数**: query (关键词) 和 city (城市) 必须提供
2. **可选参数**: salary、experience、degree、scale 只在有值时才会添加到URL中
3. **简化输入**: 支持简洁的中文输入，自动转换为正确代码
4. **参数验证**: 配置生成器会验证输入并给出警告提示

现在你可以告别手动查找代码的烦恼，用最简洁的方式生成配置文件！
